# pyDSvDCAPI

Python library for the DSvDC API.

## Installation

```bash
pip install pyDSvDCAPI
```

## Usage

```python
import pyDSvDCAPI

print(pyDSvDCAPI.__version__)
```

## Development

Install the package in editable mode:

```bash
pip install -e .
```

Run tests:

```bash
python -m pytest tests/
```
